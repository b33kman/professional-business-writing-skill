# Contributing to Professional Business Writing Skill

Thank you for your interest in contributing! This skill helps professionals write clearer, more authoritative content by removing AI-generated patterns.

## How to Contribute

### Found an AI Pattern We're Missing?

If you notice a common AI writing pattern that this skill doesn't catch:

1. Open an issue describing the pattern
2. Include before/after examples
3. Explain where you've seen this pattern

### Improving Instructions

The skill is only as good as its instructions. If you notice:
- Unclear guidance
- Missing context
- Better examples

Submit a pull request with improvements to `SKILL.md`.

### Testing

Before submitting changes:
1. Test the skill with real AI-generated content
2. Verify it maintains professional tone
3. Check that it doesn't over-correct into casual language

## Pull Request Process

1. Fork the repository
2. Create a branch: `git checkout -b improve-pattern-detection`
3. Make your changes
4. Test thoroughly
5. Commit: `git commit -m "Add detection for X pattern"`
6. Push: `git push origin improve-pattern-detection`
7. Open a Pull Request

## Style Guidelines

When editing `SKILL.md`:
- Keep instructions clear and actionable
- Use before/after examples
- Maintain professional tone
- Test changes with real content

## Questions?

Open an issue or reach out via GitHub discussions.

## Code of Conduct

Be respectful and constructive. We're all here to improve professional writing.
